# Suivi de présence intelligent en temps réel
Projet académique réalisé à l'ISET.

## Description
Ce projet consiste à développer un système de suivi de présence intelligent basé sur un capteur biométrique.  
L’objectif est de remplacer le registre papier par une solution moderne, rapide et fiable.  
L’élève marque sa présence par empreinte digitale, l’enseignant valide via une application mobile, et l’administration accède aux données en temps réel à travers un dashboard connecté à une base de données.  
En cas d’absence, les parents sont automatiquement notifiés.

## Matériel utilisé
- ESP32
- Capteur d’empreinte digitale (R307)
- Câblage et alimentation

## Fonctionnement
1. L’élève pose son doigt sur le capteur biométrique pour marquer sa présence.  
2. L’empreinte est vérifiée automatiquement puis envoyée à l’application mobile de l’enseignant.  
3. Les informations de présence sont stockées dans une base de données et consultables en temps réel dans un dashboard destiné à l’administration.  
4. Si l’élève est absent, une notification est automatiquement envoyée aux parents.

## Réalisation
- Programmation du microcontrôleur
- Intégration du capteur biométrique  
- Développement ou intégration d’une application mobile pour l’enseignant  
- Création d’un dashboard connecté à une base de données  
- Tests du système en temps réel
